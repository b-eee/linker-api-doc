

#### GetFile

添付ファイルデータの取得

**Description**

添付ファイルデータをダウンロードします

**Method**

GET

**Request URL Format**

```text
/api/v0/files/:file-id
```

**URL Params**

```text
file-id    : ファイルID
```

**Request URL Sample**

```text
GET https://api.xxx.com/api/v0/files/5a43c3c4fbfcba5c7443d037
```

**Response Sample**

```text
(省略）※該当ファイルのバイナリデータをダウンロード
```
